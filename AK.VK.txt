No.9 (Theory)



Loop in C++

Loops in C++ are used to execute a block of code repeatedly until a certain condition is met.
they help in manage repetitive tasks


1. For loop
used when the number of iterations is known before head.



2. While loop
used when the number of iterations is not known and depends on condition



3. Do-while loop
similar to the while loop , but guarantees that the block of code will be executed at least once
before checking the condition.
